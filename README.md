# Java Loop Programs

This repository contains Java programs demonstrating the use of different looping constructs.

## Contents

### Do-While Loop Programs

* Armstrong Number
* Factorial
* Palindrome Number
* Reverse Number
* Sum of Digits
* And other do-while examples

### For Loop Programs

* Number Patterns
* Multiplication Tables
* Factorial
* Fibonacci Series
* Prime Number
* And other for-loop examples

### While Loop Programs

* Fibonacci Series
* Sum of Numbers
* Reverse Number
* Palindrome Number
* And other while-loop examples
